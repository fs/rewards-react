# rewards-react
FE app for AWESOME project called REWARDS

## Setting up

### Pre-requirements

- NVM
- Node 11

```bash
nvm install 11 && nvm use 11
```

- yarn

```bash
brew install yarn
```

### Resolve dependencies

```bash
yarn install
```

### Run checks

```bash
yarn lint
```

### Run tests

```bash
yarn test
```

### Run app

```bash
yarn start
```
